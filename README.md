# Analyse-des-donn-es-CTIS-20260312
Analyse du dataset CTIS : preprocessing robuste, modélisation supervisée, évaluation des performances et visualisation des corrélations. 

Analyse complète du dataset CTIS visant à explorer les facteurs associés aux résultats d’essais cliniques.
Le projet inclut : nettoyage des données, construction d’un pipeline de preprocessing robuste (imputation, encodage, sélection automatique des variables), entraînement d’un modèle de classification, évaluation détaillée des performances et visualisation des corrélations via heatmap annotée.
Les résultats montrent un fort déséquilibre de classes et un faible pouvoir prédictif des features, ouvrant la voie à des améliorations via rééquilibrage, modèles avancés et feature engineering.

Analyse des open datas accessible ici: https://euclinicaltrials.eu/search-for-clinical-trials/?lang=en
