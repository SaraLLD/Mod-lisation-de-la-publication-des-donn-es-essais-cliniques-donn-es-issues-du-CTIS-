# Analyse-des-donn-es-CTIS-20260312
Analyse du dataset CTIS : preprocessing robuste, modélisation supervisée, évaluation des performances et visualisation des corrélations. Le projet met en évidence un fort déséquilibre de classes et un faible signal prédictif, ouvrant la voie à des améliorations via feature engineering et modèles avancés.
